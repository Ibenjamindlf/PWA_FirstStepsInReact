# FirstStepsInReact-App

This repository contains the development of the **Mandatory Practical Assignment: React** for the subject **Programación Web Avanzada (PWA)**.

The objective of this project is to build a web application using the React library that functions as a personal manager for movies and TV series.

## Project Overview

The application allows users to manage a list of movies and series they want to watch or have already watched. Users can add new items, mark them as watched, edit their information, and delete them with confirmation.

Each item includes the following attributes:

- Title
- Director
- Release year
- Genre (selected from a dropdown list)
- Rating
- Type (Movie or Series)

The application displays two separate lists:

- Content to watch
- Watched content

## Functional Requirements

The application must include:

- A counter displaying the total number of items per list and per genre
- A search input to filter items by title or director
- Filters by genre and type
- Sorting options by year and rating (ascending and descending)
- A message indicating when a list is empty
- A message indicating when no results match the applied filters
- Persistent data storage using **localStorage**, ensuring data remains after page reload

## Technologies Used

- React (main library)
- Vite (development environment and build tool)
- JavaScript (ES6+)
- CSS
- npm (package manager)
